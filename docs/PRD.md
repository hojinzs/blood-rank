# PRD: 혈부심 — 혈액형 팀전 헌혈 독려 웹사이트

**버전:** v0.5
**작성일:** 2026-03-10
**상태:** 검토 중

---

## 1. 제품 개요

### 한 줄 정의
혈액형별 헌혈 수급 현황을 실시간으로 보여주고, 혈액형 성격설을 활용해 경쟁심을 자극하는 헌혈 독려 웹사이트

### 배경
대한적십자사 bloodinfo.net은 혈액형별 보유일수(A형 X.X일, B형 X.X일…)를 실시간으로 공개하지만, 이 데이터는 무미건조한 표 형태로 제공된다. 한편 한국에는 혈액형 성격설에 대한 강한 문화적 친숙함이 있다. 이 두 가지를 결합하면 — "내 혈액형 팀이 꼴찌네?" — 자연스러운 바이럴과 헌혈 동기 유발이 가능하다.

---

## 2. 문제 정의

### 현재 상황
- 혈액 부족은 상시 발생하지만, 일반인이 현재 어느 혈액형이 부족한지 직관적으로 인식하기 어렵다.
- 기존 헌혈 캠페인은 "생명을 살려요" 식의 무거운 메시지에 집중 → 젊은 층에게 행동 유발력 약함.
- 혈액형 성격설은 여전히 밈·대화 소재로 소비되지만 헌혈과 연결된 적이 없다.

### 기회
- 가벼운 경쟁심("우리 팀 꼴찌야?")이 진지한 설득("헌혈하세요")보다 공유율이 높다.
- 실시간 데이터가 있으므로 매일 달라지는 순위가 재방문 유인이 된다.
- SNS 공유 포맷("나 O형인데 지금 1등 수호 중 🩸")으로 자연 확산 가능.

## 5. 핵심 기능 (MVP 범위)

### 5-1. 혈액형 팀 순위판 (메인 화면)
- 4개 혈액형(A/B/O/AB)을 팀으로 표시
- 보유일수 기반 순위 정의 (확정)
  - **1위 = 보유일수 높음 = 헌혈 많이 한 팀 = 칭찬 대상**
  - **꼴찌 = 보유일수 낮음 = 혈액 위기 = 저격 대상**
- 각 혈액형에 고유 색상/슬로건 부여 (캐릭터 없음 — 확정)
- 순위가 실시간으로 반영됨 (데이터 갱신 주기: 1회/일 또는 가능 시 더 자주)

**순위 판정 기준 예시:**
- 보유일수 5일 이상: 🟢 "여유 있음 (자만하지 마)"
- 3~5일: 🟡 "위태위태"
- 3일 미만: 🔴 "위기! 팀이 지고 있어"
- 1일 미만: 🚨 "긴급 구조 요청"

### 5-2. 혈액형 성격 + 수급 연동 카피
수급 상태에 따라 해당 혈액형의 성격설을 비틀어 조롱/응원하는 카피를 표시.

| 혈액형 | 성격 특징 | 수급 부족 시 카피 예시 |
|--------|-----------|----------------------|
| A형 | 꼼꼼, 완벽주의, 책임감 강함 | "꼼꼼한 A형이 이렇게 무책임할 리가요? 지금 바닥인데요." |
| B형 | 자유분방, 자기중심적, 개성 강함 | "B형답게 내 마음대로 한다면서요. 그럼 헌혈도 내 마음대로 해줘요." |
| O형 | 사교적, 열정적, 리더십 강함 | "O형이 리더라며요. 팀이 꼴찌인데 리더가 뭐해요?" |
| AB형 | 이중적, 독특, 합리적 | "AB형은 합리적이잖아요. 지금 헌혈이 합리적인 선택입니다." |

수급 여유 시에도 다른 톤으로 카피 존재:
- "지금 1위예요. 자만하면 안 되겠죠?"

### 5-3. 내 혈액형 선택 & 공유
- 유저가 자신의 혈액형 선택 → 맞춤 카피 + 순위 확인
- SNS 공유 카드 생성 — **이미지 카드 방식으로 확정**
  - Vercel OG로 동적 생성 (혈액형 색상 배경 + 현재 순위/일수 + 한 줄 카피)
  - 예시 카드 문구: "O형 현재 꼴찌 🩸 2.8일치 / 혈부심 지키러 간다"
  - 피드 썸네일로 눈에 띄는 것이 핵심
- 공유 목적지: 트위터(X), 카카오스토리, 인스타 스토리

### 5-4. 헌혈 CTA
- 각 혈액형 상태 카드 하단에 "지금 헌혈 예약하기" 버튼
- bloodinfo.net 헌혈의 집 예약 페이지로 연결

### DB 스키마 (Supabase)

#### `blood_supply_daily` — 일별 스냅샷 (핵심 테이블)

```sql
CREATE TABLE blood_supply_daily (
  id           uuid        PRIMARY KEY DEFAULT gen_random_uuid(),
  date         date        NOT NULL,                   -- KST 기준 날짜
  scraped_at   timestamptz NOT NULL,                   -- 실제 수집 시각
  blood_type   text        NOT NULL CHECK (blood_type IN ('A','B','O','AB')),
  days         numeric(4,1) NOT NULL,                  -- 보유일수 (예: 3.5)
  rank         smallint    NOT NULL CHECK (rank BETWEEN 1 AND 4),
                                                       -- 1 = days 높음 = 칭찬
                                                       -- 4 = days 낮음 = 저격
  status       text        NOT NULL,                   -- 'good'|'ok'|'warning'|'critical'
  created_at   timestamptz DEFAULT now(),

  UNIQUE (date, blood_type)
);
```

**status 임계값 (스크래핑 시 계산)**
| status | 조건 | 화면 표시 |
|--------|------|----------|
| `good` | days ≥ 5.0 | 🟢 여유 있음 |
| `ok` | 3.0 ≤ days < 5.0 | 🟡 보통 |
| `warning` | 2.0 ≤ days < 3.0 | 🔴 위태위태 |
| `critical` | days < 2.0 | 🚨 긴급 |

---

#### `blood_supply_streaks` — 연속 순위 기록 (명예의전당용, MVP 이후)

```sql
CREATE TABLE blood_supply_streaks (
  id             uuid     PRIMARY KEY DEFAULT gen_random_uuid(),
  blood_type     text     NOT NULL CHECK (blood_type IN ('A','B','O','AB')),
  rank_position  smallint NOT NULL CHECK (rank_position IN (1, 4)),
                                      -- 1 = 1위 연속, 4 = 꼴찌 연속
  streak_start   date     NOT NULL,
  streak_end     date,               -- NULL = 현재 진행 중
  streak_length  int      NOT NULL DEFAULT 1,
  is_current     boolean  NOT NULL DEFAULT true,

  CHECK (streak_end IS NULL OR streak_end >= streak_start)
);
```

---

#### View: `blood_supply_latest` — 오늘 + 전일 대비 변화

```sql
CREATE VIEW blood_supply_latest AS
SELECT
  t.blood_type,
  t.date,
  t.days,
  t.rank,
  t.status,
  t.scraped_at,
  t.days - y.days        AS days_delta,   -- 전일 대비 보유일수 증감
  t.rank - y.rank        AS rank_delta    -- 전일 대비 순위 변화 (음수 = 상승)
FROM blood_supply_daily t
LEFT JOIN blood_supply_daily y
  ON y.blood_type = t.blood_type
  AND y.date = t.date - INTERVAL '1 day'
WHERE t.date = CURRENT_DATE;
```

`rank_delta`로 "어제보다 2계단 상승" 같은 내러티브 카피 자동 생성 가능.

### 스크래핑 & 저장 플로우

```
[Vercel Cron] 매일 07:00 KST
  → GET /api/cron/scrape  (CRON_SECRET 헤더 검증)
      │
      ├─ fetch bloodinfo.net 보유량 현황 페이지
      ├─ HTML 파싱 → { A: 3.5, B: 5.4, O: 2.8, AB: 5.3 }
      │
      ├─ rank 계산 (days DESC → rank 1~4 부여)
      ├─ status 계산 (임계값 기준)
      │
      ├─ Supabase UPSERT blood_supply_daily
      │     ON CONFLICT (date, blood_type) DO UPDATE
      │     (같은 날 재실행 시 덮어쓰기)
      │
      ├─ streak 업데이트 (is_current 레코드 갱신)
      │
      └─ Next.js ISR revalidate 트리거
            → / 페이지 재생성
```

**실패 처리**
- 파싱 실패 시 Supabase에 기록 없음 → 프론트는 마지막 성공 데이터 표시 + "X일 전 데이터" 안내
- Vercel Cron 실패 알림: Vercel Dashboard 로그 또는 간단한 Slack Webhook

## 8. 콘텐츠 방향

### 전체 톤 & 매너
- 무겁지 않되 장난스럽지도 않은 **"귀여운 도발"** 톤
- **중요 컨텍스트:** 혈액형 성격설은 2020년대 이후 MBTI에 밀려 "진지하게 믿는 문화" 아님 → **추억의 밈/아이러니**로 소비하는 세대가 타깃. 진지함 배제 필수.
- 혈액형 성격설을 믿는다고 전제하지 않음 — 어디까지나 **밈으로서** 소비
- 혈액형 차별·진지한 성격 판단 등 민감한 방향으로 가지 않도록 주의
- 헌혈을 "해야 한다"가 아니라 **"이왕이면 네가 하면 어때?"** 뉘앙스 유지
- 부작용 방어: 면책 문구 대신 **"혈액형 성격설은 과학적 근거가 없습니다. 그냥 재밌잖아요 😄"** 한 줄로 처리 (확정)

### 사이트 이름 & 타이틀 구조 (확정)

**브랜드명:** 혈부심 (고정)
**도메인:** 별도 결정 (배포 시 확정)
**페이지 타이틀 포맷:** `혈부심 | {1위 혈액형 동적 카피}`

```
예시)
혈부심 | A형은 역시 다정다감해
혈부심 | B형이 정말 이기적일까?
혈부심 | O형은 역시 열정이 달라
혈부심 | AB형의 남다름은 여기서도
```

`<title>` 태그 및 OG title 모두 동일 포맷 적용. 링크 공유 시 미리보기 제목이 매일 달라지는 효과.

**동작 원리:** 1위(보유일수 높음 = 헌혈 많이 한 팀) → 긍정 고정관념 칭찬 / 꼴찌(보유일수 낮음 = 위기) → 부정 고정관념 저격

| 혈액형 | 1위 동적 타이틀 | 꼴찌 카드 카피 |
|--------|--------------|--------------|
| A형 | A형은 역시 다정다감해 | "A형이 이렇게 소심할 줄이야. 헌혈은 그냥 가면 되는데 왜 눈치 봐요?" |
| B형 | B형이 정말 이기적일까? | "B형은 역시 자기 피만 아끼는군요." |
| O형 | O형은 역시 열정이 달라 | "리더라면서요. 팀이 꼴찌인데 리더는 뭐해요?" |
| AB형 | AB형의 남다름은 여기서도 | "AB형 오늘은 또 이런 날이군요. 기분 탓인가요?" |

### 순위별 카드 멘트 전체 매트릭스

#### A형 (긍정: 다정다감·꼼꼼·책임감 / 부정: 소심·눈치·완벽주의 강박)
| 순위 | 카드 카피 |
|------|----------|
| 🥇 1위 | "역시 A형. 꼼꼼하게 챙길 건 다 챙기네요. 오늘은 인정합니다." |
| 🥈 2위 | "A형 오늘 선방했어요. 근데 1위한텐 좀 배워요, 배려심 그쪽이 더 많은 거 아닌가요?" |
| 🥉 3위 | "A형 3위예요. 이러다 꼴찌 됩니다. 눈치 보는 거 여기선 하지 마요." |
| 🔴 꼴찌 | "A형이 이렇게 소심할 줄이야. 헌혈은 그냥 가면 되는데 왜 눈치 봐요?" |

| 순위 | 카드 카피 |
|------|----------|
| 🥇 1위 | "의외라고요? 아니요. B형은 하기로 하면 합니다. 오늘 그 날이에요." |
| 🥈 2위 | "B형 2위. 1위도 할 수 있는데 왜 딱 거기서 멈춰요. 자기 페이스 타다가?" |
